It also allows the user to discover trending anime on the discover page, as well as to search for anime they would like to add to their watch list.

Critique Code

Critique Code takes the good parts of Reddit and Stackoverflow and mashes them together. Tired of getting personally attacked for asking a question on Stackoverflow? Critique Code solves that. Critique Code is meant to be a place where new or veteran developers can come and discuss the thing that they share in common: An interest in code. Critique Code doesn't take it's self too seriously, and we expect the users to also share the same ideal. Make jokes, show intentionally bad code. Just have a good time!
